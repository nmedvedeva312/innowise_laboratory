# Lecture 3: Student Grade Analyzer

This program is a console-based application to manage and analyze student grades.  

## Features

- Add a new student
- Add one or more grades for a student
- Generate a full report with:
  - Each student's average grade
  - Max, min, and overall average grades
- Find the top student
- Handles invalid inputs and edge cases

## How to Run

Make sure you are in the `lecture_3` folder with the virtual environment activated.

```bash
python main.py
